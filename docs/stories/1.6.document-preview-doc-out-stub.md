# Story

As a non‑tech user,
I want to preview markdown documents and see a doc‑out placeholder,
so that I can validate content before export.

# Acceptance Criteria

1. Preview markdown từ mẫu/tệp giả lập
2. `doc‑out` hiển thị placeholder (sẽ nối BE ở Epic 2)
